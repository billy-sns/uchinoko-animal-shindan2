## 📌 プロジェクト概要

### サービス名
**うちの子アニマル診断（四柱推命版）**

### コンセプト
2歳〜未就学児の子どもを持つママパパ向けに、「うちの子のトリセツ」として子どもの性格・気質を診断するサービス。

### 算命学版から四柱推命版への切り替え理由
算命学版では判定結果が実際の性格と「タイプ自体がズレる」違和感があった。四柱推命の十二運星をベースにした判定では、検証結果がかなり良好に出たため切り替えを決定。

- ページ最下部に「※四柱推命をベースに独自アレンジ」と注記
- 「動物占い」「5アニマル」「個性心理學」は一切使わない
- 60キャラの細分類は使わない

## 🔮 診断ロジック（四柱推命版）

### ロジック方針
- **採用：十二運星（12種）で1対1マッピング**
- **流派：五行同根説**
  - 戊（つちのえ）は丙（ひのえ）と同じ運行
  - 己（つちのと）は丁（ひのと）と同じ運行

### 計算の3ステップ

```
Step A：生年月日 → 日柱（日干＋地支）を算出
　↓
Step B：日干＋地支 → 十二運星を判定
　↓
Step C：十二運星 → 12動物にマッピング
```

## 📐 Step A：日柱の計算ロジック

### 基本アイデア
- 60干支は60日周期で循環する
- 基準日（干支がわかっている日）からの経過日数を計算
- 経過日数 ÷ 60 の余りから60干支を割り出す

### 基準日
```
1900年1月1日 = 甲戌（きのえいぬ）
　→ 60干支リストの11番目
　→ 0始まりインデックスでは10
```

### JavaScriptでの実装例

```javascript
// 基準日：1900年1月1日 = 甲戌 = 60干支の11番目（インデックス10）
const BASE_DATE = new Date(1900, 0, 1);
const BASE_GANSHI_INDEX = 10;

function calcDayPillar(year, month, day) {
  const targetDate = new Date(year, month - 1, day);
  
  // 経過日数を計算（ミリ秒 → 日に変換）
  const diffMs = targetDate - BASE_DATE;
  const diffDays = Math.floor(diffMs / (1000 * 60 * 60 * 24));
  
  // 60干支のインデックスを計算（0〜59）
  // 過去日付の場合のためのマイナス対策付き
  const ganshiIndex = ((BASE_GANSHI_INDEX + diffDays) % 60 + 60) % 60;
  
  // 日干（天干10種）と地支（12種）を分解
  const dayKanIdx = ganshiIndex % 10;
  const dayShiIdx = ganshiIndex % 12;
  
  return { ganshiIndex, dayKanIdx, dayShiIdx };
}
```
